Project Overview

This application predicts the next five 5-minute SPY stock price candles using multiple neural network models. Each model is trained on randomly selected historical data and achieves 90% accuracy or higher independently.

  • Real-time SPY data collection
  • 10 distinct neural network models
  • Each model trains on random monthly data
  • Accurate predictions for the next 5 candles
  • Visualization of predictions and uncertainty

How It Works

  1. Download Data: Collect the most recent SPY 5-minute candles
  2. Train Models: Train 10 distinct neural networks on different historical data
  3. Make Predictions: Generate forecasts for the next five 5-minute candles
  4. Visualize Results: View predictions with confidence intervals

Model Architecture

Model 1: Bayesian Neural Network

Incorporates MC Dropout, Gaussian HMM, and Viterbi algorithm for probabilistic modeling and uncertainty estimation.

Model 2: Bozdogan Consistent AIC

Integrates fuzzy logic and stochastic processes to balance model complexity and market noise.

Model 3: Viterbi Algorithm + Baum-Welch

Uses Bayesian priors, fuzzy logic for transitions, and state sequence modeling.
